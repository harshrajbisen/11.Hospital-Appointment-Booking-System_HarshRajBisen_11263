# 🏥 Hospital Appointment Booking System

A simple **Hospital Appointment Booking System** developed in **Core Java** using **Object-Oriented Programming (OOP)** concepts. This console-based application allows users to manage doctors, patients, and appointments efficiently while preventing double booking of time slots.

## 📌 Features

- 👨‍⚕️ Manage doctor information
- 🧑‍🤝‍🧑 Manage patient information
- 📅 Book appointments
- ❌ Cancel appointments
- 📋 View all appointments
- ⏰ Prevent duplicate bookings for the same doctor and time slot
- 💻 Menu-driven console interface

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner (Java Input)

## 📂 Project Structure

```
HospitalAppointmentBookingSystem.java
```

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/HospitalAppointmentBookingSystem.git
```

2. Navigate to the project folder.

```bash
cd HospitalAppointmentBookingSystem
```

3. Compile the Java file.

```bash
javac HospitalAppointmentBookingSystem.java
```

4. Run the program.

```bash
java HospitalAppointmentBookingSystem
```
